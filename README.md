# FlameCourt｜炎上法庭

**FlameCourt｜炎上法庭** 是一款以網路輿論、公共討論與資訊判讀為主題的網頁遊戲。玩家將扮演「公民稽核員」，在高度混亂的流量環境中維持理性、蒐集證據、保護無辜者，並透過程序與規範完成最終任務。

本作品採虛構世界觀與諷刺表現形式，適合作為網頁互動作品、媒體素養教材、公共議題展示頁或遊戲概念展示。

---

## 遊戲特色

- **輿論攻防玩法**：面對洗版、情緒化留言、標籤化攻擊與資訊煙霧。
- **蒐證推進關卡**：依序取得時間軸、商業透明、創作尊重、動員證據與平台規範五項核心證據。
- **風險控制機制**：玩家不只要抵抗攻擊，也必須避免誤傷無辜者，防止監督行動變成獵巫。
- **Boss 戰結局判定**：依據理性值、公信力、獵巫值與證據完整度，進入不同結局。
- **純靜態網頁**：不需要後端伺服器，可直接部署於 GitHub Pages。

---

## 遊戲目標

玩家需要依序完成五個蒐證關卡，取得五項核心證據：

1. 時間軸核心
2. 商業透明核心
3. 創作尊重核心
4. 動員證據核心
5. 平台規範核心

集滿五項核心證據後，將進入最終關卡「炎上法庭」，與失控流量 Boss 進行最終對決。

## 手機遊玩

手機或平板開啟後，畫面下方會出現觸控按鈕，可直接操作移動、跳躍與技能。建議使用橫向螢幕遊玩，以取得較完整的畫面比例。

## 操作方式

| 操作 | 功能 |
| --- | --- |
| `←` / `→` 或 `A` / `D` | 左右移動 |
| `↑` / `W` / `Space` | 跳躍 |
| `Z` | 冷靜光束，清除留言砲火 |
| `X` | 事實盾牌，短時間抵擋傷害 |
| `C` | 透明炸彈，清除場上威脅並提升公信力 |
| `R` | 重新開始 |
| 手機觸控按鈕 | 移動、跳躍、冷靜光束、事實盾牌、透明炸彈 |

---

## 狀態系統

| 數值 | 說明 |
| --- | --- |
| 理性值 | 玩家承受攻擊的能力，歸零即失敗 |
| 公信力 | 代表玩家行動的正當性與說服力 |
| 獵巫值 | 代表誤傷無辜、過度攻擊或失控反制的風險 |
| Boss 血量 | 最終關卡的主要勝利條件 |

玩家若只追求攻擊而忽略比例原則，可能導致獵巫值過高並進入失敗結局。

## 線上部署

本專案可直接部署到 GitHub Pages。

### GitHub Pages 部署步驟

1. 建立新的 GitHub repository。
2. 將本專案所有檔案上傳到 repository 根目錄。
3. 開啟 repository 的 `Settings`。
4. 進入 `Pages`。
5. Source 選擇 `Deploy from a branch`。
6. Branch 選擇 `main`，資料夾選擇 `/root`。
7. 儲存設定後，等待 GitHub Pages 產生網站連結。

專案內亦包含 GitHub Actions workflow，可用於 GitHub Pages 自動部署。
